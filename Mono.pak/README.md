# Mono

A minimalist MP3 player for Trimui Brick running NextUI/MinUI.

## Features

- Browse and play MP3, FLAC, OGG, and WAV files
- Simple, battery-friendly dark interface
- ID3 tag support for metadata display
- D-Pad navigation

## Controls

| Button       | Action                |
| ------------ | --------------------- |
| **D-Pad ↑↓** | Navigate list         |
| **D-Pad ←→** | Seek -10s / +10s      |
| **A**        | Select / Play-Pause   |
| **B**        | Back / Exit           |
| **L / R**    | Previous / Next track |
| **Start**    | Menu                  |

## Installation

1. Copy `Mono.pak/` to `/Tools/tg5040/` on your SD card
2. In NextUI, go to **Tools > Mono**

## Music Location

Place your music files in `/Music/` on your SD card.
